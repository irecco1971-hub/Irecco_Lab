# Irecco Lab — Produkty

## MillWork

**Branża:** Stolarstwo CNC  
**Model:** B2C — plugin do SketchUp, sprzedaż bezpośrednia / Extension Warehouse  
**Stack:** Ruby (SketchUp Ruby API) + HTML/JS (HtmlDialog)  
**Repozytorium:** github.com/irecco1971-hub/millwork  
**Target launch:** Spring 2027

Plugin do SketchUp dla stolarzy i producentów mebli CNC. Automatyzuje pipeline od modelu 3D do danych maszynowych: wycena, lista cięć, eksport DXF/CSV. Zastępuje godziny ręcznego przeliczania i przepisywania danych. Rynek: małe i średnie warsztaty, które nie stać na drogie oprogramowanie CAM.

**Aktualny etap:** PROJECT STONE — finalizacja (ST-05 + ST-07 backend gotowe, czekają UI)

## TechHub

**Branża:** Szkolenia B2B  
**Model:** B2B custom, single-tenant (klient: Aluprof S.A.)  
**Stack:** Python 3.12 + FastAPI + HTML5 + Claude API  
**Repozytorium:** github.com/irecco1971-hub/techhub  

Narzędzie do produkcji i dystrybucji materiałów szkoleniowych dla Aluprof. Przyjmuje PPTX, konwertuje do PDF/MP4/SCORM/Flipbook, nakłada branding, wzbogaca przez AI (quiz, streszczenie, tłumaczenie) i eksportuje do LMS HCM DECK.

**Aktualny etap:** Faza 3 — AI Studio (backend 5/5 gotowy, trwa implementacja UI)

## Thomos

**Branża:** Zdrowie / dokumentacja medyczna  
**Model:** B2C bezpłatny dla pacjenta + B2B instytucje (szpitale, ubezpieczyciele, pracodawcy)  
**Stack:** Next.js 16 + React 19 + Claude API → docelowo Bielik (polski LLM)  
**Repozytorium:** github.com/irecco1971-hub/thomos  
**Target:** Beta publiczna Q4 2026

Platforma zdrowotna — tłumacz i organizator dokumentacji medycznej napędzany AI. Pacjent wgrywa wyniki badań, recepty, skierowania; Thomos tłumaczy z medycznego na ludzki, kategoryzuje i buduje historię zdrowotną. Punkt odniesienia: vital.io (5M+ użytkowników USA) — Thomos buduje ekwiwalent dla rynku EU.

**Aktualny etap:** MVP beta — refaktoryzacja Next.js (Etap 3 ✅, następny: Etap 4 widoki)
