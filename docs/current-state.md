# Irecco Lab — Current State

## Current Phase
Foundation planning.
The project is being designed from zero as a modular, documentation-first platform for multiple future products under Irecco Lab.

## Immediate Goal
Create a strong but minimal foundation that supports the first revenue-oriented product without overbuilding the platform.

## Current Product Priority
MillWork is the first anchor product.
It should drive real architectural decisions instead of hypothetical future products.

## Current Architecture Decision
- Python modular monolith
- Open-source-first stack
- Claude Code as primary implementation environment
- Git-based workflow
- Session-close auto-commit, but no auto-push

## Next Steps
1. Finalize documentation package
2. Create repository structure
3. Initialize core backend skeleton
4. Add MillWork as first product module
5. Build product entry flow: landing, beta access, docs, feedback
